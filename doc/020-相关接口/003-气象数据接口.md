# 气象数据接口  
以下接口仅供爱好者测试，不可用于商业产品使用。  
如需商业使用，请联系点灯团购购买。  

## 天气数据  
更新频率：每2小时更新一次  
接口：
```
https://iot.diandeng.tech/api/v2/weather/{city}?device={deviceName}&key={authKey}
```
响应：
```json
{
    "city": "成都市",
    "province": "四川",
    "weather": "阴",
    "temp": "28",
    "humi": "45",
    "wind": "东南",
    "power": "≤3",
    "updateTime": "2020-05-16 20:27:42"
}
```
