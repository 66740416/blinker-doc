# blinker轻松物联网  
blinker是一个物联网接入方案，旨在让大家可以轻松畅快地DIY物联网设备。
- 其由服务器端、app端、设备端组成，可以部署到几乎所有物联网平台  
- app端支持ios、android  
- 设备端可以使用蓝牙、WiFi、MQTT等方式接入  
- 通过界面布局器，用户可自己拖拽布局设备控制界面，自由打造你的物联网设备
